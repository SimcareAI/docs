Before you start adding students, take a few minutes to configure FieldX for your program. Navigate to **My Program** from the navigation menu.

![Customization Overview](/images/customization-1.png)

## Organization settings

![Organization Settings](/images/customization-2.png)

### Edit info

Update your program name, domain, and address.

### Academic terms

Academic terms control how student progress is tracked across a semester. Here you can:

- Set the start and end dates of the current term
- Create multiple terms for cohorts running on different schedules

### Student tracks

Student tracks define hour requirements. You can:

- Enable and disable tracks (e.g., remote vs. in-person) — this determines what students see during the clearance process
- Set total, direct, indirect, and supervision hour requirements per track

## Clearance / application process

The clearance process determines what students must complete before they can log hours. By default, students are asked to:

- Select their track
- Review and sign standard agreements
- Select at least one site and supervisor (or request a placement)
- Upload a liability insurance form

![Liability Form Upload](/images/customization-3.png)

Programs commonly customize this to:

1. Require additional documents (background checks, site affiliation forms, expiration dates)
2. Change the required agreements
3. Remove the site selection requirement from initial clearance

![Clearance Configuration View](/images/customization-4.png)
