All paperwork in FieldX — forms, evaluations, hour-log reports, agreements — lives in the **Assignments** tab on your main dashboard. This is the program-wide template library; once created, anything here can be assigned to any cohort or individual.

![Assignments Tab Overview](/images/create_assign_1.png)

## The Assignments tab

The tab has two primary actions in the top right:

- **New Group** — create a folder to bundle related forms together (e.g., "Mid-term evaluations")
- **Create Assignment** — open the assignment builder

Filter the table by **Type** (`All Types | Group | Form | Report`), creator, and cohort. With groups in use the table columns are `Name | Type | Created By | Assigned to Cohort | Actions`; without groups it's `Name | Type | Created By | Contents / Cohorts | Actions`.

![Main Dashboard Interface](/images/create_assign_2.png)

## Creating an assignment

Click **Create Assignment** to open the assignment builder. It has three tabs:

### Form

Two ways to build a form:

- **Build New** — start with a blank canvas and drag-and-drop fields in the [form builder](/fieldx/building-forms)
- **Upload** — *Convert document to form with AI*. Upload an existing PDF or Word doc (e.g., your current Mid-Term Evaluation) and SimCare parses it into a FieldX form, with a live progress indicator while it runs

### Timesheet (Report)

A **Report** in FieldX is the schedule for logging hours — when hours are submitted and who needs to sign them.

![Creating a Report Template](/images/create_assign_3.png)

Configure:

| Setting | Description |
|---|---|
| **Title & description** | Clear name like "Weekly Hour Log" or "Month 1 Summary" |
| **Student signature required** | Switch — whether the student signs |
| **Supervisor signature required** | Switch — whether the site supervisor signs |
| **Faculty signature required** | Switch — whether the faculty supervisor signs |

<Tip>You don't have to require a student signature — some programs only collect supervisor approval.</Tip>

### Templates

A grid of built-in templates (`BUILTIN_TEMPLATES`) you can use as a starting point — each has a **Use template** action.

## Assigning a form or report

Click the actions menu (⋯) on any row and choose **Assign**.

![Report Actions Menu](/images/create_assign_4.png)

The **Assign** modal has two tabs:

- **Create** — make a new assignment from this template
- **Current** — view existing assignments using this template

In the Create tab, pick the assignment target:

- **Cohorts** — assign to one or more cohorts (all students in those cohorts get it)
- **Individuals** — assign to specific people (ideal for targeted evaluations)

![Assignment Pop-up](/images/create_assign_5.png)

Then set a due date and (optionally) make it recurring. See [Setting up recurring tasks](/fieldx/setting-up-recurring-tasks) for the full schedule options.

If a student has a conflicting assignment (e.g., same form already assigned), the modal surfaces a conflict dialog so you can resolve it.

## Forms vs. reports

| Type | Use for |
|---|---|
| **Forms** | Evaluations, surveys, agreements, custom questionnaires — Google-Forms-like with HIPAA-compliant storage |
| **Reports** | Hour logs and supervision sign-off forms — multi-signer DocuSeal workflow |

## Form creation options

Forms can be built two ways:

![Form Creation Options](/images/create_assign_6.png)

- **Build New** — drag-and-drop in the form builder
- **AI import** — upload a PDF or Word document and let AI convert it

You can also bundle multiple forms and reports into a **Group** (using the **New Group** button) and assign the whole group at once.

## Per-row actions

Each row in the table has actions for:

- **View** — preview the form/report
- **Assign** — assign to cohorts or individuals
- **Regenerate form** — re-run AI parsing if the source doc has changed
- **Add to group** / **Manage group**
- **Delete** (single or bulk)

## Editing an existing assignment

To change just the due date of an already-emitted assignment (or to adjust who's on a multi-signer form), open the assignment row inline and click **Edit**. The dedicated edit page lets you change the due date and, for multi-person assignments tied to a cohort, exclude specific student/supervisor pairs.

<Note>For details on building complex forms, see [Building forms](/fieldx/building-forms). For recurring assignments, see [Setting up recurring tasks](/fieldx/setting-up-recurring-tasks).</Note>
